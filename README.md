# 🗺️ GIS Portfolio

A collection of spatial analytics, geoprocessing workflows, and mapping projects.  
This portfolio showcases my work across terrain modelling, hydrology, spatial ETL, and Python-based GIS automation.

---

## 🌍 Focus Areas

- Terrain modelling (DEM, LIDAR, hillshade, slope, aspect)
- Hydrology & flow modelling (flow direction, accumulation, catchments)
- Spatial ETL pipelines (CRS standardisation, geometry validation, PostGIS workflows)
- Python geoprocessing (GeoPandas, Rasterio, Shapely)
- QGIS & ArcGIS Pro automation
- Spatial visualisation & cartographic outputs

---

## 📂 Repository Structure

- **01-projects** — complete GIS analyses with data, scripts, notebooks, and outputs  
- **02-tools** — reusable Python utilities, QGIS scripts, and ArcGIS toolboxes  
- **03-visuals** — maps, layouts, and screenshots  
- **04-docs** — methodology notes, diagrams, and templates
